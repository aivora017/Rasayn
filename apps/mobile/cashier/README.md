# @pharmacare/mobile-cashier

**Cashier App** — Android tablet billing — voice-first, barcode camera, counter-side workflow.

## Status
**SCAFFOLD** — Expo SDK 52 + React Native 0.76 + Reanimated 3.16. Ready for `expo install` and EAS build.

## Setup
```sh
cd apps/mobile/cashier
npm install
npx expo start
```

## Build
```sh
eas build --platform android --profile preview
eas build --platform ios --profile preview
```
