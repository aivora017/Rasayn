# @pharmacare/mobile-customer

**Customer App** — Refill, loyalty, family vault, side-effect read-aloud, ABHA-linked record.

## Status
**SCAFFOLD** — Expo SDK 52 + React Native 0.76 + Reanimated 3.16. Ready for `expo install` and EAS build.

## Setup
```sh
cd apps/mobile/customer
npm install
npx expo start
```

## Build
```sh
eas build --platform android --profile preview
eas build --platform ios --profile preview
```
