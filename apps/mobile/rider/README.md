# @pharmacare/mobile-rider

**Rider App** — Last-mile delivery — pickup, route, signature, COD reconcile via Shiprocket+Dunzo.

## Status
**SCAFFOLD** — Expo SDK 52 + React Native 0.76 + Reanimated 3.16. Ready for `expo install` and EAS build.

## Setup
```sh
cd apps/mobile/rider
npm install
npx expo start
```

## Build
```sh
eas build --platform android --profile preview
eas build --platform ios --profile preview
```
