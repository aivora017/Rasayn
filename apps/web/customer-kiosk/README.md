# @pharmacare/web-customer-kiosk

**Customer Kiosk** — Voice-first kiosk in-store. Elderly-friendly. 'Is my Crocin ready?' in Hindi.

## Status
**SCAFFOLD** — Next.js 15.5 + React 19.2 + App Router. Cloudflare Workers runtime target.

## Setup
```sh
cd apps/web/customer-kiosk
npm install
npm run dev
```
