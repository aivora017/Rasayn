# @pharmacare/web-distributor-portal

**Distributor Portal** — Pharmarack-class B2B portal for distributors. Catalog, orders, schemes, settlements.

## Status
**SCAFFOLD** — Next.js 15.5 + React 19.2 + App Router. Cloudflare Workers runtime target.

## Setup
```sh
cd apps/web/distributor-portal
npm install
npm run dev
```
